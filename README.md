# Excel Gird

## Technology Used
Type Script, Html, Css

## Features 
- draw the header row, column and cell

- support the 1,00,000 row and 500 columns

- selection of row, column and range

- add text in to the cell and also support undo redo inside input box

- add row and column resizing

- undo redo functionality in row, column resize and the writen text

- escape key to cancle editing 

- selection of range, row and column via shift + arrowkey

- move the focus of cell using arrow key

- reach at end of row column using ctrl + arrow key

- enter key use to save the cell text and move to below cell

- direct type in cell make the cell active

- f2 key and double click to edit the cell


## How to Run
Add Node Package and compile the typescript files

```
mkdir dist
npm install
npm run build:watch
run the live server to run html
```
